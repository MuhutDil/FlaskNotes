# FlaskNotes
This app works with notes.

The back end is implemented on Flask using SQLAlchemy and SQLite.

The front end is implemented in JavaScript without using frameworks. Creates an interactive single-page application with HTML, CSS, and JavaScript.

This is a training project based on an article [RealPython](https://realpython.com/flask-javascript-frontend-for-rest-api/).

#### Build the application and preparation for launch
    python -m pip install -r requirements.txt
#### Starting server
    make start
#### Populate the database with prepared data
    make db
